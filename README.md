Projeto de visualização de dados referentes aos Airbnbs de Seattle usando Tableau

Este projeto foi realizado com base no curso do software Tableau disponível no canal do YouTube Alex The Analyst, conforme o link abaixo:

https://www.youtube.com/@AlexTheAnalyst

Link do guia para o projeto:

https://www.youtube.com/watch?v=zOR0-nygfDE

Link para a base de dados:

https://www.kaggle.com/datasets/alexanderfreberg/airbnb-listings-2016-dataset

Este projeto foi realizado com o intuito de aplicar conceitos de análises de dados e aplicá-los
para confeccionar painéis de visualização e transmitir informações relevantes sobre um tema em específico.

Durante o processo foram testadas diferentes formas de visualização, formatação e alocação dos gráficos e tabelas no dashboard de forma que a sua compreensão se dê da maneira mais intuitiva possível.
